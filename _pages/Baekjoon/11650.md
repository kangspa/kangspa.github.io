import sys
input = sys.stdin.readline

if __name__ == '__main__':
    N = int(input())
    coordiate = [tuple(map(int, input().split())) for _ in range(N)]
    # 파이썬 정렬 함수는 문제 요구사항을 그대로 따라준다.
    for x, y in sorted(coordiate): print(x, y)

<h2>문제</h2>

<div class="problem-text" id="problem_description">
<p>2차원 평면 위의 점 N개가 주어진다. 좌표를 x좌표가 증가하는 순으로, x좌표가 같으면 y좌표가 증가하는 순서로 정렬한 다음 출력하는 프로그램을 작성하시오.</p>
